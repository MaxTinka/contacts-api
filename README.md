# Contacts API

A RESTful API for managing contact information, built with Node.js, Express, and MongoDB. This is Part 1 of a two-week project focusing on retrieving contact data.

## 🚀 Live Demo

- **Base URL:** https://contacts-api-111111.onrender.com
- **All Contacts:** https://contacts-api-111111.onrender.com/contacts
- **Single Contact:** https://contacts-api-111111.onrender.com/contacts/{id}

## 📋 Features (Part 1)

- ✅ Connect to MongoDB Atlas database
- ✅ GET all contacts endpoint
- ✅ GET single contact by ID endpoint
- ✅ Error handling for invalid IDs (404)
- ✅ Environment variable security
- ✅ Deployed on Render

## 🛠️ Tech Stack

- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB Atlas
- **ODM:** Mongoose
- **Deployment:** Render
